In this repository I will collect different office hacks to do extremely boring stuff, like making slides a bit faster

### Google Slides Scripts

In [this](https://github.com/vifirsanova/extremely-boring-stuff/tree/main/google_slides_scripts) folder you can find scipts for changing font settings in Google Slides. Just open Extentions > Apps Sripts > Copy, paste to their console, change my scipts if needed, then run them > Brilliance and flawless victory
